# thoth-cli

Divination tools for the terminal: astrology, tarot, gematria, numerology.

- **Astrology**: Swiss Ephemeris (NASA-grade planetary calculations)
- **Tarot**: `secrets.SystemRandom` (true cryptographic entropy)
- **Gematria**: Hebrew, Greek, English classical systems
- **Numerology**: Pythagorean with master numbers

## Quick Start

```bash
npm install -g thoth-cli
thoth --help
```
